# Experiment 04: Real Public Data Integration

**Status:** Planned  
**Phase:** 2  
**Engine:** Shared Data

## Objective

Integrate public Thai and international data sources that improve realism, validation, and downstream feature quality.

## Planned Sources

- HDC prevalence and epidemiology
- NHSO open data
- Thai e-GP procurement
- FDA Orange Book refresh validation
- ClinicalTrials.gov where relevant

## Method Placeholder

Document for each source:

1. Source description and access path
2. Field mapping into project schema
3. Cleaning and normalization steps
4. Coverage and completeness checks
5. Sanity-check comparison against external reports

## Evaluation Placeholder

- Row and field completeness
- Missingness by source and year
- Cross-source consistency checks
- Sanity checks against known external summaries

## Required Artifacts

- Data mapping tables
- Quality report
- Output file paths
- Limitations and licensing notes
