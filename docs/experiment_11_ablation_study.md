# Experiment 11: Ablation Study

**Status:** Planned  
**Phase:** 4  
**Engine:** Forecasting / Integration

## Objective

Measure the value of each major feature group and signal source by removing them systematically.

## Planned Ablations

- Epidemiology anchors
- Budget / utilization features
- Patent and exclusivity signals
- RAG-extracted regulatory events
- Competitive / substitution features

## Planned Evaluation

- Delta in forecast metrics
- Which features help only specific classes
- Robustness across model families
